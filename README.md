# weather_app

A weather app built using the Flutter framework. It is an adaptive UI that will change according to the OS (iOS/Android).

## Features

- The weather API was integrated to show real-time weather and forecasted weather.
- Icons change dynamically according to the weather.
- A loading screen is used to allow time for the API details to load properly.
- ListView allows the user to check up to 10 weather cards.
