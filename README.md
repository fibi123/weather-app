# weather_app

A weather app built using the flutter framework. It is an adaptive UI that will change according to the OS (iOS/Android). 

##Features
-The weather API was integrated to show real-time weather and forecasted weather. Icons are changeable according to the weather as well. 
-A loading screen is utilized to allow time for the details from the API to be loaded properly. 
-ListView is also added to allow the user to check upto 10 weather cards.
